# Restaurant Tipping Behavior Analysis
### Exploratory Data Analysis (EDA) using Python

This project analyzes the 'tips' dataset to identify patterns in customer behavior. As part of my Data Science journey (currently 47% through CodeWithHarry's course), I used this project to practice data manipulation and visualization.

## Key Features
* **Data Cleaning:** Handled categorical variables like 'day', 'sex', and 'smoker'.
* **Feature Engineering:** Used **NumPy** (`np.where`) to create a 'is_generous' flag for tips > 15%.
* **Visualization:** Created **Violin Plots** to compare bill distributions and **Regression Plots** to show the correlation between total bill and tip amounts.

## Tech Stack
* **Language:** Python (Anaconda/Jupyter)
* **Libraries:** Pandas, NumPy, Seaborn, Matplotlib
